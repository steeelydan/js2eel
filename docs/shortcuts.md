## Keyboard Shortcuts

### Editor

| Linux & Windows | Mac             | Description                  |
| --------------- | --------------- | ---------------------------- |
| `Ctrl + S`      | `Cmd + S`       | Save                         |
| `Ctrl + I`      | `Cmd + I`       | Autoformat                   |
| `Ctrl + /`      | `Cmd + /`       | Toggle line comment(s)       |
| `Alt + Up/Down` | `Alt + Up/Down` | Move line(s) up/down         |
| `Ctrl + D`      | `Cmd + D`       | Select next symbol occurence |
| `Ctrl + F`      | `Cmd + F`       | Open search panel            |
